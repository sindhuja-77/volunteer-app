Project Overview
This project is a simple web application built as part of the Teach For India Web Development Intern pre-work.

The application allows volunteers to:
Register using a two-step form
Provide personal details and availability preferences
Submit their information for review by Teach For India staff

An optional admin view is provided to view registered volunteers.

Features
Login and Signup flow (UI-based)
Two-step volunteer registration form
Navigation header across all pages
Clean and responsive UI
Admin dashboard to view registered volunteers
Backend API to store volunteer data (optional task)
